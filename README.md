# Heat-Sterilization-Calc
Heat-Sterilization-Calc （pH・温度連動型 加熱殺菌シミュレーター）食品加工におけるpH値と加熱温度に基づき、安全な殺菌に必要な推奨加熱時間を自動算出するアプリケーションです。

HACCP管理の効率化と、科学的根拠に基づいた加熱工程の設定を支援します。


🍯 Heat-Sterilization-Calc

（pH・温度連動型 加熱殺菌シミュレーター）

食品加工におけるpH値と加熱温度に基づき、
安全な殺菌に必要な推奨加熱時間を自動算出するアプリケーションです。

HACCP管理の効率化と、科学的根拠に基づいた加熱工程の設計をサポートします。

🌿 概要

Heat-Sterilization-Calc は、食品加工現場における加熱殺菌条件を
シンプルにシミュレーションできるツールです。

🧪 pH値と加熱温度を入力

⏱ 推奨加熱時間を自動算出

📋 現場での工程設計をサポート

✨ 特徴
📊 pH × 温度によるシンプルな計算ロジック

🧾 HACCP管理を意識した設計補助

🧑‍🍳 現場オペレーションを想定した軽量設計

🍃 余計な機能を削ぎ落とした実務特化型ツール

🚀 使い方
アプリを起動します
🧪 pH値を入力します
🌡 加熱温度（℃）を入力します
▶️「計算」ボタンを押します
⏱ 推奨加熱時間（時間：分）の目安に基づき、設定時間を調整し、ボツリヌス菌増殖の恐れのない安全圏の確認ができます。


🍓 想定用途
ジャム・シロップ・加工食品の加熱工程設計

小規模食品製造現場での基準づくり

📋 HACCP記録の補助資料

🧪 試作レシピの安全性確認


⚠️ 注意事項

本アプリは、加熱条件設計のための補助ツールです。

食品の安全性を保証するものではありません。

最終的な加熱・殺菌条件の判断は、

各自治体の基準・専門家の助言・製品特性に基づいて行ってください。
